                            README.TXT


1，基本情况
本压缩包用于将计算机上安装的龙驭视频会议系统2.0的客户端设置为使用视音频文件模拟
真正的麦克风和摄像头。

2，使用方法
将本压缩包中的文件解压缩到客户端的安装目录，覆盖相应的文件即可。
注意：客户端的配置文件将被覆盖，您在其中的部分设置（保存的用户名、密码等将会丢失）

3，文件说明

    AVClips/example.wav：模拟的音频文件
    AVClips/video.dat：模拟的视频文件
    conf/Flvcc.ini：新的配置文件

    注意：不要使用其他文件替代上述音频文件和视频文件。
